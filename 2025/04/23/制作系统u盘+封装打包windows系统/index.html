制作系统u盘+封装打包windows系统

发表于 2025-04-23 分类于 windows Valine： 本文字数： 558 阅读时长 ≈ 1 分钟

制作系统盘

工具准备

准备一个U盘以及以下工具

  • Ventoy
  • 微PE工具箱
  • UltraISO
  • VMware Workstation 虚拟机

安装Ventoy

下载好解压压缩包后，打开Ventoy2Disk.exe，然后对要做系统盘的U盘进行格式化安装Ventoy

打开系统盘，看到有一个ventoy的文件夹就说明安装成功了，安装成功后在需要新建一个存放想要安装系统ISO的文件夹，我这里就新建的名为ISO文件夹

然后打开VentoyPlugson.exe，这个插件是用来设置ventoy的，确认是安装ventoy的u盘，点击启动会自动弹出网页，没弹出网页的话就手动点链接进去

插件设置中还有很多功能，比如更改主题，设置bios启动ventoy的壁纸等，可以自己去研究

点击左边列表的全局控制插件，我这里喜欢都设置为中文，看个人喜好，需要设置指定搜索目录

看U盘是哪个盘符，设置搜索目录为该盘符下的我们创建的ISO目录，点确定保存，出现有效就设置成功了

这时候就可以把我们喜欢的系统ISO放进U盘内了

制作微PE系统ISO

下载好微PE工具箱后打开，我们不安装，只需要生成可启动的ISO，放进ventoy系统盘中

下载windows官方镜像

推荐几个下载镜像的途径

  • 微软官网
  • MSDN旧版、MSDN新版
  • 系统库
  • 系统之家
